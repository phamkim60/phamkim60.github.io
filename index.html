Réalisations

Projet "Brico-Déco"

Plateforme collaborative pour la recherche et le partage de tutoriels de bricolage et de décoration.

Backend: API REST en Symfony 7 avec base de données MySQL et ORM Doctrine, intégrant un back-office léger via Twig pour gérer des entités spécifiques.

Frontend: Application web monopage (SPA) réactive et responsive en React 18, avec gestion d'état via Redux Toolkit et requêtes AJAX via Axios.

Projet de fin d'études réalisé en équipe (4 développeurs) sur 20 jours.
